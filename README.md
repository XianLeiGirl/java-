# java-
快捷注释：ctrl+\
允许拼接的写法：字符串+字符串，字符串+数字，混合拼接
变量和数值是可以混合运算的
boolean变量不能加减乘除
不支持从boolean到int的转换
隐式转换：编译器认为从int到double是没有损失的，相当于显示转换
int b = 1;
int a = b++;则a=1,b=2（先赋值再自加）
int a = ++b;则a=2,b=2（先自加再赋值）
逻辑操作符：&&逻辑与，||逻辑或，!逻辑非
进行与、或、非的运算变量必须均为boolean类型
关系表达式的值为boolean类型
整型的除法运算结果不保留小数部分
double型的除法运算结果保留小数部分
